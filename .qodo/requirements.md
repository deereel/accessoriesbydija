## Functional Requirements

### Customer Storefront
- [x] Product catalog browsing
- [x] Product search and filtering
- [x] Shopping cart
- [x] Checkout process
- [x] Customer account management
- [x] Order history
- [x] Newsletter subscription

- [x] Dashboard with key metrics
- [x] Product management (CRUD)
- [x] Order management
- [x] Customer management
- [x] Inventory management
- [x] Category management
- [x] Banner management
- [x] Testimonial management
- [x] Shipping rate management

- [x] PWA support for offline access

- [x] Secure authentication

- PHP 8.x+
- MySQL 5.7+
- HTTPS support
- Service Worker support (for PWA)
